# React Form Builder

A powerful, flexible, and user-friendly React-based form builder with drag-and-drop functionality. Build complex forms with ease using an intuitive visual editor.

## Features

- **Drag-and-drop Interface**: Intuitive form builder with drag-and-drop components
- **Rich Field Types**: Support for text, number, email, date, checkbox, radio, select, and more
- **Validation**: Built-in validation with Zod schema validation
- **Export Options**: Export forms to PDF, JSON, or HTML
- **Modern UI**: Beautiful, modern UI using Radix UI components and Tailwind CSS
- **TypeScript**: Full TypeScript support for type safety

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
